McDropdownTrigger

This directive is intended to be used in conjunction with an mc-dropdown tag. It is responsible for toggling the display of the provided dropdown instance.

Selector: [mcDropdownTriggerFor]

Exported as: mcDropdownTrigger
Properties
Name Description
@Input('mcDropdownTriggerData')

data: any

Data to be passed along to any lazily-rendered content.
@Input('mcDropdownTriggerFor')

dropdown: McDropdownPanel<any>

References the dropdown instance that the trigger is associated with.
@Output()

dropdownClosed: EventEmitter<void>

Event emitted when the associated dropdown is closed.
@Output()

dropdownOpened: EventEmitter<void>

Event emitted when the associated dropdown is opened.

dir: Direction

The text direction of the containing app.

opened: boolean

Whether the dropdown is open.

openedBy: 'mouse' | 'touch' | null

McDropdownDefaultOptions

Default mc-dropdown options that can be overridden.

Properties
Name Description

backdropClass: string

Class to be applied to the dropdown's backdrop.

hasBackdrop: boolean

Whether the dropdown has a backdrop.

overlapTriggerX: boolean

Whether the dropdown should overlap the dropdown trigger horizontally.

overlapTriggerY: boolean

Whether the dropdown should overlap the dropdown trigger vertically.

The x-axis position of the dropdown.

yPosition: DropdownPositionY

The y-axis position of the dropdown.

    type DropdownPositionX = 'before' | 'after';
